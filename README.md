# Parque Natural

Proyecto de Bases de Datos II centrado en el diseño y la implementación de un sistema de gestión para un parque natural.

## Scripts principales

- `ScriptTablas.sql`: crea la estructura de la base de datos.
- `ScriptDatos.sql`: inserta los datos iniciales.

## Ejecución

1. Ejecutar `ScriptTablas.sql` para crear la base de datos.
2. Ejecutar `ScriptDatos.sql` para cargar los datos iniciales.
